# グラインディッシュ 技術プロフィール v1.2（制作資料）

**注記：演出用の文芸表現・引用は含めない。事実と運用のみ記載。**

## 0) 取扱方針（最重要）

* **位置づけ**：破壊＝再編のトリガー。乱暴者ではなく、構造を壊して再配置する工匠。
* **好感の源泉**：①壊す前に内部構造を**見せる** ②**安全手順**で実演 ③**後始末**を徹底。
* **直接暴力**：**非敵生体への直接打撃は全面禁止**。ただし **王のフルーツ（イ・セイ・スプレムーティ）およびその随員・実行部隊等の敵対者には、直接打撃を許可**。

  * **運用**：**合図なし／溜めなし／即時攻撃**。主武器＝**オッキオマッツァ（魚骨ハンマー）**。
  * **表現強度**：**残虐表現（欠損・過剰流血）の強度は、放送局の規定に準ずる**。
* **放送乱相（グリッチ）**：意図しない症状。**1話あたり 1–2回／各10–30秒**まで（理解阻害を避ける）。

## 1) 概要（事実のみ）

* **肩書**：工匠・科学者（緑の一族／Verd-Linia）。ジューストニウム発見に関与。
* **存在状態**：過去に死亡扱い。現代期は**バックアップ生体データの抽象個体**（本人と完全同一視しない）。
* **召喚経路**：ドマロン内「**デスマスクの会議**」経由。現世実体化は**ブルーモンスター、ブルーアイズ**への一時憑依（prefab置換）。
* **反応傾向**：

  * **王のフルーツおよび従者**：**即時の直接打撃**で応答（予告なし）。
  * **非敵（一般生体・知性体）や果実一般**：**生体打撃に置換しない**（象徴物・構造物破壊で処理）。

## 3) 外見・所作（晩年＝復活時の姿）

復活時の肉体は死亡時のデータから再構成されるため、以下は晩年の姿であり、復活後の姿でもある。

* **髪**：**白髪主体、わずかに緑がかった色が混在**。肩にかかる長さのくせ毛が四方に跳ねる。ボリュームがあり、後頭部に向かって膨らむ（後頭部に渦）。
* **顎髭**：**白い顎髭**。口元から顎全体を覆う程度の長さ。整えてはいるが几帳面ではない。
* **肌**：**日焼けした褐色**。皺は深いが、頬と目元に笑った跡の皺が刻まれている。
* **目**：琥珀色。**オレンジサングラス**常用（遮光・グリッチ緩和）。**オレンジのフレーム**はやや大振りで丸みを帯びた形状。母メガネのオレンジサングラスとの視覚的継承関係が明確。
* **服**：**緑ツナギ（ジャンプスーツ）**。全身を覆うワンピース型。柄は**葉脈模様**——濃淡の異なる緑の葉が全面に密集したパターン（遠目には迷彩のようにも見えるが、近づくと一枚一枚が葉であるとわかる）。インナーは**黒のTシャツ**。ベルトは**黒革、銀のバックル**。
* **靴**：**黄緑色のスニーカー**。ローカット。鉄プレートは外見からは見えず内蔵。
* **体格**：痩せ身・約175cm。腕が長く手が大きい。**筋力は人並み**。
* **所作**：起立前に**床を2回軽打**（開始合図）。口元に軽い笑み。
* **印象**：ヴィリナに「ダサい緑の服」と言われるのが完全に納得できるビジュアル。本人は気に入っている。陽気そうに見えるが、サングラスの奥の目は見えない。

### 母メガネとの視覚的継承

* 緑の服 ← メガネも緑のワンピースを着用
* オレンジサングラス ← メガネも同型のオレンジサングラス
* 息子は母親と同じ色を纏って荒野を走っていた

### 3-b) 外見（青年期＝荒野の運び屋時代・約142〜120年前）

* **髪**：**鮮やかな緑と白のツートーン**。晩年より緑が濃く強い。白髪はまだ少なく、緑が主体。短めだが上方向に激しく跳ねる。
* **髭**：なし。顔は若く、痩せて角張っている。
* **肌**：日焼けした褐色。砂埃と果汁の汚れが常についている。
* **サングラス**：**オレンジ（赤寄り）のフレーム**。晩年と同型。母メガネから引き継いだもの。若い顔にやや大きく見えるが、それが彼のアイデンティティ。
* **服**：**オレンジのTシャツ**＋**緑のパンツ**。晩年の緑ツナギとは異なり上下で色が分かれている。Tシャツにはグラフィックプリント。荒野の過酷な生活で**服は常にボロボロ**。腕にブレスレットやバンドを複数。
* **体格**：痩せ身で細い。まだ筋肉もつききっていない青年の体。晩年と同じく腕が長く手が大きい。

| 要素 | 青年期 | 晩年（＝復活時） |
|---|---|---|
| 髪色 | 緑が主体、白が少量 | 白が主体、緑がわずかに残る |
| 髭 | なし | 白い顎髭 |
| サングラス | 同型オレンジ | 同型オレンジ（変わらない） |
| 服の色 | オレンジT＋緑パンツ（上下分離） | 緑ツナギ（全身一体） |
| 服の状態 | ボロボロ、汚れだらけ | きれい（復活時に再構成） |
| 体格 | 細い青年 | 痩せた老人 |
| 印象 | 尖っている。怒りと熱がある | 穏やかに見える。笑みの下は読めない |

> **【制作注記：色の移行】** 青年期はオレンジ（母の色＝果汁の色＝敵の色）を纏いながら緑（自分の色）を下に着る。晩年は全身が緑に統一される——母の色を脱ぎ、自分の色だけになった。あるいは、戦う対象を自分の上に着る必要がなくなった。

## 5) 二相状態（症状セット）

**原因**：ジューストニウム過剰曝露の後遺。**本人の意志ではない**。

### 5.1 静基相（Basa-Quieta）

* 物静か・判断が速い・短文。

### 5.2 放送乱相（Emitto-Glitcha）

* **主トリガ**：疲労／人混み／強光・フラッシュ／果実・果実族の接近／糖分過多（スムージー等）
* **症状（演出可・乱用禁止）**：

  1. **【BEEP】**：発話の一部が自動消音（台本は【BEEP】表記／本人の聴取理解は保持）
  2. **顔相偏移**：一瞬だけ別人・犬・老女等に置換
  3. **フレーム飛び**：半拍ズレが周囲にも波及
  4. **音楽混入**：環境に短尺ダンスミュージック。必要時のみ**ダンスホール演出**を挿入、即収束
  5. **生体模写**：猿・汽笛・女性の喘ぎ声等の口真似（必要時のみ）
* **危険度**：対人攻撃性は上がらないが**判断誤差**増。対応＝**三歩距離／短文対話／注視回避**。
* **緩和**：オレンジサングラス／**4-2-6呼吸**／水・塩飴／**ドマロン**退避。

## 6) 能力と限界

* **身体**：平均的。握力は年齢比やや強め。スタミナ短距離型。
* **現場合成（ブリコラージュ）**：廃材・工具で即席解。

  * 段取り：①見取り（資材・接合の弱強マップ）→②結び論理（梃子・摩擦・結束）→③仮留（可逆固定）→④小テスト→本実装。
* **破壊技**：腕力ではなく**順序と力学**（例：衝撃→裂皮→脱核）。

  * **対敵対者**：同じく**段取り重視**。**初撃は最短動線・振りかぶり無し**。
* **限界**：関節痛／息切れ／グリッチ再発。**若者の肉体には勝てない**。

## 8) ツール（機能記述）

* **魚骨ハンマー／オッキオマッツァ**

  * **構成**：魚頭蓋・脊椎骨格。握把に振動伝達管。腹部に微量不活性化液層。尾部**安定鰭**で反動制御。
  * **意味／思想**：監視の“目”を砕く象徴具。**過度連打不可**（反動・疲労）。
* **作業槌（小）／ブリコルハンマー**

  * **用途**：現場合成。柄に「くそじじい（セノブラッカ）の落書き（ヴィリナ）。
* **トラック（クー）**

  * **構成**：母メガネの飼い猫**クー**（ミューズの生物）をコアに、荒野の廃材をブリコラージュして組み上げた車両。
  * **声**：関西弁の機械声。メガネの傍で育った猫の言葉を宿す。「物に声を宿す」技術の原点。
* **保護具**：サングラス／薄手エプロン／軍手。

* **ヴィリナ**：呼び出しは**チムリー（チャイムの精）通知後**。役割分担＝彼女が**道を作る**／本人が**古い障害を壊す**。半友好的。

## 11) 年代記（抜粋）

* **幼少〜少年期（沈黙紀後期・約160〜142年前）**：極地ミューズ施設で誕生・教育。10歳で造神衆の襲撃により両親を失う。施設に8年間残り、文明の記録・ミューズの観測技術・緑の一族の機械技術を吸収。復讐のために力を蓄えた期間。
* **青年期（沈黙紀後期・約142〜125年前）**：18歳、三つの動機（①外への探求の教え ②緑の一族の使命 ③復讐心）で施設を出る。母の飼い猫**クー**をコアに廃材からトラックを組み上げ、運び屋として活動。22歳でスオ（虫族）と出会い、ジューストニウムを発見。23歳でオッキオ・サト祖先と合流し四人チーム完成。約12年間のジューストニウム採掘事業。「荒野の魔術師」の異名。
* **壮年期（沈黙紀末期〜再生紀・約125〜115年前）**：35歳で採掘事業が軌道に乗り離脱。造神衆との攻防。聖杯再起動。コマツの予言。仲間離散。ヴェジスタンへ逃亡し妻と娘をもうける。
* **晩年（再生紀・約115〜100年前）**：単身で大坂へ。表向きは「街の迷惑老人」、裏では地下室で研究。幻想論文**「命令詩の成立と果汁変異紀」**を執筆。ジューストニウムによる未来シミュレーション。くまちゃん読書会との交流。ドマロン建設（約103〜101年前）。道具たち・ブルーアイズの創造。60代前半で死亡、意識は4体のブルーアイズに分散保存。
* **復活期（現代紀・Verd-Legenda Vioda）**：ヴィリナのドマロン到来をトリガーに復活。土曜朝の演目。**解説 → 実演 → 片づけ**が流儀。
